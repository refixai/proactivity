## What each example demonstrates

- **LangGraph** (`examples/langgraph`) ŌĆö a signup-welcome agent on `createReactAgent`: `fromLangGraph(agent)`, `governed(sendEmail)` as the one structural change, `langchainModel(llm)` for reflection, `pinned: true` goal seeding, and a short local-development cadence.
- **Anthropic SDK** (`examples/anthropic`) ŌĆö the same agent on `anthropicLoop`: `anthropicLoop({ client, model, tools, system })`, `governed: true` on the outbound tool, and `anthropicModel(client, model)` for reflection.

## Run one locally

Both examples share the same setup. Pick the adapter you want:

<CodeGroup>
```bash LangGraph
cd examples/langgraph
pnpm install
cp .env.example .env  # add ANTHROPIC_API_KEY
pnpm dev
```

```bash Anthropic
cd examples/anthropic
pnpm install
cp .env.example .env  # add ANTHROPIC_API_KEY
pnpm dev
```
</CodeGroup>
