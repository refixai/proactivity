---
title: "Introduction"
description: "The TypeScript SDK for proactive agents: durable wake scheduling, cross-wake goal memory, LLM-driven cadence, and idempotent action governance."
---

`@refix/proactivity` is a TypeScript SDK that turns an agent you already have into a proactive one. You wrap it in a single call; from then on the agent wakes on its own schedule, carries what it learned into the next wake, sets its own cadence within bounds you define, and routes side effects through a governance envelope that enforces idempotency and rate caps.

The agent's reasoning, tools, and prompt do not change. The SDK briefs the agent before each wake, records what it does, and reflects afterward to update memory and choose the next wake time.
